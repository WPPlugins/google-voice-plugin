== Description ==

The Google Voice Widget plugin allows WordPress sites to add a Google Voice Widget to anyplace they have widgetized on their site. Also, more features to come soon!


Feature List:

* Basic cut and paste Google code snippet to activate widget
* Single widget for use

Version .02

* Added Customizable Call Me badge - Text or HTML

Version .021
* Fixed a path name bug.

Version 0.2.2

== Installation ==

How to install the plugin and get it working.

e.g.

1. Upload uncompressed gvwidget folder to the `/wp-content/plugins/` directory or upload from WordPress.org Extend directory via the 2.8+ version interface.
2. Activate plugin.
3. Profit. Paste generated code snippet from your Google Voice Widget generator into the Google Voice Plugin settings page. Optionally, add text or HTML to the widget.
4. Go to widgets and add Google Voice Widget to any widgetize area on your site.
5. Set hours that you do not want the widget to allow calls.
6. Save the widget.
7. All done!
